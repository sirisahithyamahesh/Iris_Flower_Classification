# Iris Flower Classification

This project uses Machine Learning to classify Iris flower species using Scikit-learn.

## Algorithms Used
- Random Forest Classifier

## Dataset
Iris dataset from Scikit-learn

## Features
- Sepal Length
- Sepal Width
- Petal Length
- Petal Width

## Accuracy
Achieved around 95-100% accuracy.

## Technologies
- Python
- Google Colab
- Scikit-learn
- Pandas
